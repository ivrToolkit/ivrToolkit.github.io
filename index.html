Nuget packages

ivrToolkit.core

The core API which replaces the old ivrToolkit.Base project. Includes the system voice files and a voice.properties file you need to set up that defines what plugin you want to use.

ivrToolkit.Plugin.Dialogic.Sip

A plugin implementing SIP using the Dialogic HMP 3.0 up service update 525. It has been tested on Microsoft Servier 2019 and Hyper-V.

ivrToolkit.Plugin.Dialogic.Analog

A plugin for use with Dialogic analog boards such as the d/120jct-ls. It's been tested with the Dialogic System release 6.0 build 271.

ivrToolkit.Plugin.Simulator

Intended for use with your computer sound card and the ivrToolkit.VirtualPhone. For testing your projects. I recomment using one of the dialogic plugins instead for your testing if you can.
