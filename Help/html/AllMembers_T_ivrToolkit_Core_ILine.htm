ILine MembersA Sandcastle Documented Class Library

The ILine type exposes the following members.
Methods
  NameDescription
CheckStop
Use this within long computational methods to check if the line has hungup or stopped.
Close
Closes the line and releases the resources.
Dial
Dials a phone number using call progress analysis.
FlushDigitBuffer
Returns every character including the terminator
GetDigits
Keep prompting for digits until number of digits is pressed or a terminator digit is pressed.
Hangup
Forces a hangup on the line.
PlayFile
Plays a wav file which must be in the format of 8000hz 1 channel unsigned 8 bit PCM.
RecordToFile
Records a wav file to the disk in the format of 8000hz 1 channel unsigned 8 bit PCM.
Stop
Call this to stop the line and release all its resources.
TakeOffHook
Pick up the line.
WaitRings
The number of rings to wait before answering

Extension Methods
  NameDescription

(Defined by LineExtensions.)
PlayDate
Plays a DateTime object given based on the mask parameter.
(Defined by LineExtensions.)
PlayInteger
Same as PlayNumber but for an integer.
(Defined by LineExtensions.)
PlayMoney
Speaks a number in money format. For example 5.23 would be spoken as 'five dollars and twenty three cents'
(Defined by LineExtensions.)
PlayNumber
Speaks out a number. For example 5.23 would be spoken as 'five point two three'
(Defined by LineExtensions.)
PlayOrdinal
Plays a number from 1 to 31. Example 31 would speak 'thirty first'.
(Defined by LineExtensions.)
PlayPhoneNumber
Plays a phone number as long as it is 7, 10 or 11 characters long.
(Defined by LineExtensions.)
PlayString
Plays a collection of one or more string parts which are separated with a comma. Each string part is in the format of 'data|code'. You would string them together like: 'data|code,data|code,data|code' etc.
(Defined by LineExtensions.)
