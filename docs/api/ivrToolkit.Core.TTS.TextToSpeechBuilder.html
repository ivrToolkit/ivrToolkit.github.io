GetWavStreamAsync(CancellationToken)

Gets a WavStream object representing the text message or wav file. This method generates a wav file and a txt file if the wav file name exists. Will not do TTS and save the file unless the file doesn't yet exist, or it does exist but the txt file representing the last text message is now different from the text message. TTS happens always if the wav file name is null.
